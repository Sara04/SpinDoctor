PURPOSE

DESCRIPTION 

 create initial data for solving PDE
 
 Input:
     1. ncell
     2. cell_shape
     3. params_domain_geom is a structure with 3 elements:
         Rratio_IN

     4. params_domain_pde is a structure with 8 elements:
         dcoeff_IN
